# VALAWAI-SR

Shared code for the VALAWAI social robot components.
This contains code to handle the RabbitMQ communication, shared variables, and message data classes.

## Installation

You can either copy snippets of code from here, or install this library in editable mode:

```bash
git clone https://github.com/giubots/valawai-sr your-path-here
pip install -e your-path-here
```
